============
Requirements
============

-------
Summary
-------

~~~~~~~~~~~~~
In a computer
~~~~~~~~~~~~~

On a modern operating system all you need is Nix.
You can install it by following the instructions: https://nixos.org/download

~~~~~~~~~~~~~~~~~~~
In a CI environment
~~~~~~~~~~~~~~~~~~~

All you need is support to execute Container Images

~~~~~~~~
Hardware
~~~~~~~~

* CPU: 1+ vCPU
* RAM: 4GiB+ RAM
* Disk space: 10GiB
