= REQ.221 Desconectar periféricos innecesarios

== Requisito

Una máquina virtual no debe tener conectados dispositivos
que no sean necesarios para su funcionamiento
(+Floppy+, +IDE+, +CD/DVD+, +USB+, +Serial+).
