= REQ.293 Gestionar ejecución de código en stack

== Requisito

El sistema operativo debe usar técnicas
para evitar ejecución de código en `stack`
(`ASLR`, `PaX`, `canaries`, deshabilitar `bit` de ejecución, etc).
